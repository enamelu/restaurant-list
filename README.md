# 餐廳清單 restaurant-list
此專案提供使用者查詢餐廳的資訊，例如:餐廳類別、地址、評分、描述等
目前正在開發中，只能使用localhost

##Features - 產品功能
* 依照餐廳名稱及餐廳類別搜尋
* 呈現餐廳清單
* 顯示評分
* 餐廳資訊
